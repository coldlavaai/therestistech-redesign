# The Rest Is Tech - Modern B2B Website Redesign

A premium, modern redesign of The Rest Is Tech website featuring cutting-edge visual design and smooth interactions.

## Features

- **Modern Design System**: Premium visual design with glassmorphism, gradient orbs, and 8-point grid system
- **Smooth Animations**: Intersection Observer animations, parallax scrolling, and dynamic gradient effects
- **Responsive Layout**: Mobile-first design with hamburger menu and optimized breakpoints
- **Performance Optimized**: Lazy loading, CSS custom properties, and efficient JavaScript
- **Accessibility**: Semantic HTML, ARIA labels, and keyboard navigation support

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS with custom properties, OKLCH colors, and advanced animations
- **JavaScript**: Vanilla JS for optimal performance
- **Fonts**: Inter and Clash Display from Google Fonts

## Design Highlights

- Animated gradient orbs with blur effects
- Glassmorphism cards with backdrop filters
- Premium color palette using OKLCH color space
- Micro-interactions on all interactive elements
- Smooth scroll animations and parallax effects

## Sections

1. **Hero**: Eye-catching hero with animated background and stats
2. **Services**: 6 premium service cards with hover effects
3. **About**: Company information with mission statement
4. **CTA**: Call-to-action section with gradient background
5. **Footer**: Clean footer with navigation and social links

## Deployment

Deployed on Vercel with automatic deployments from the main branch.

## Local Development

```bash
# Clone the repository
git clone [repository-url]

# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```
